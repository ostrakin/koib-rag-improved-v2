# extras/ — необязательные, в приложениях НЕ используются

- `query_logger.py` (бывш. `src/logging_module.py`) — структурированный JSONL-лог
  запросов с ротацией по дате. На момент рефакторинга **не импортируется** нигде
  в коде. Сохранён здесь как готовый компонент: при желании подключите его в
  `src/rag_pipeline.py`. В сборки `koib-indexer`/`koib-server` он не попадает.
