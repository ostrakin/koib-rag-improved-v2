# Подключение VK-бота (Callback API)

1. Поднимите API так, чтобы он был доступен по HTTPS:
   `python main.py --serve --host 0.0.0.0 --port 8000`
   Для локальной отладки используйте туннель (ngrok / Cloudflare Tunnel).

2. VK → Сообщество → Управление → Работа с API → Callback API.
   - Версия API: `5.131`.
   - Адрес сервера: `https://<домен>/vk_callback`.

3. VK покажет строку подтверждения — впишите её в `.env` → `VK_CONFIRM_CODE`.
   Также заполните `VK_SECRET_KEY`, `VK_GROUP_ID`, `VK_ACCESS_TOKEN`.

4. В разделе «Типы событий» включите `message_new`.

5. Нажмите «Подтвердить» — сервер вернёт строку подтверждения и статус станет
   «Сервер подтверждён».

6. Проверка: напишите боту в личные сообщения сообщества технический вопрос.
   Команды бота: `/start`, `/help`, `/reset`, `/health`.

## Заметки
- Бот отвечает VK мгновенно строкой `ok`, а обработку запроса выполняет в фоне —
  так VK не считает обращение неотвеченным при долгой генерации.
- Дедупликация событий по `event_id` защищает от повторной обработки ретраев VK.
- Длинные ответы автоматически разбиваются на сообщения.
- Rate-limit задаётся `VK_RATE_LIMIT_PER_MINUTE` (на пользователя) и
  `VK_GLOBAL_RATE_LIMIT_PER_MINUTE` (общий).

## Разделение моделей КОИБ (v4.12)

Ответы для КОИБ-2010 (физические кнопки ДА/НЕТ) и КОИБ-2017А/Б (сенсорный
экран) различаются принципиально, поэтому бот привязывает каждый диалог к
конкретной модели и фильтрует выдачу по ней.

Как это работает для пользователя:
1. Если модель названа прямо в вопросе («…по КОИБ-2017А…»), бот её определяет,
   использует и запоминает.
2. Если модель неизвестна, бот один раз показывает inline-клавиатуру выбора
   (КОИБ-2010 / КОИБ-2017А / КОИБ-2017Б / «Не знаю / любая»), запоминает заданный
   вопрос и отвечает на него сразу после выбора.
3. Команда `/model` меняет модель в любой момент.
4. Выбор сохраняется между сообщениями в `vk_user_prefs.db` (каталог
   `METADATA_DIR`).

Технические детали:
- класс `VKUserPrefStore` хранит `model` (`koib2010`/`koib2017a`/`koib2017b`/
  `any`) и отложенный вопрос на пользователя;
- модель пробрасывается в `RAGPipeline.answer` как `model_filter` и
  `strict_model` (строгий фильтр включается флагом `MODEL_FILTER_STRICT`);
- при выборе «Не знаю / любая» жёсткий фильтр не применяется (поиск по всем
  моделям), а в шапке ответа указывается, что модель не задана.

Никаких дополнительных прав у VK-приложения для клавиатуры не требуется —
используется штатный параметр `keyboard` метода `messages.send`.
